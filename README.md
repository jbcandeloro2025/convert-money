# 💱 Currency Convert

Conversor de moedas em tempo real, desenvolvido como desafio prático do curso **DevClub**.

---

## 📋 Sobre o projeto

Aplicação web que permite converter valores entre diferentes moedas utilizando taxas de câmbio atualizadas em tempo real, consumindo a [AwesomeAPI](https://docs.awesomeapi.com.br).

---

## ✨ Funcionalidades

- Conversão entre 5 moedas: Real Brasileiro, Dólar Americano, Euro, Peso Argentino e Guarani Paraguaio
- Taxas de câmbio em tempo real via API
- Exibição da cotação atual ao selecionar as moedas
- Atualização automática da bandeira e nome da moeda ao trocar o select
- Formatação do valor de acordo com a moeda selecionada
- Layout responsivo para mobile, tablet e desktop

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- [AwesomeAPI](https://economia.awesomeapi.com.br) — cotações em tempo real

## 🚀 Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Selecione a moeda de origem e a moeda de destino
4. Digite o valor a ser convertido
5. Clique em **Converter**
